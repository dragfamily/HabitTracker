# HabitTracker

Aplikacija za praćenje navika, dostupna kao:

- 📱 **Android aplikacija** – folder: `/android`
- 🌐 **Web aplikacija** – folder: `/web`

## Folderi

| Folder   | Opis                          |
|----------|-------------------------------|
| `/android` | Android Studio projekat       |
| `/web`     | Web aplikacija (npr. React)   |
| `/shared`  | Deljeni fajlovi (API, assets) |
